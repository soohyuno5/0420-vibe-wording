# 명언 생성기 (Quote Generator)

HTML/CSS/JavaScript 기초 학습을 위한 인터랙티브 명언 생성기 웹사이트입니다.

## 📋 프로젝트 개요

이 프로젝트는 웹 개발의 기본 구조를 배우기 위해 설계된 명언 생성기입니다. 라이트/다크 테마 전환 기능과 반응형 디자인을 포함하고 있습니다.

## ✨ 주요 기능

### 🎯 핵심 기능
- **랜덤 명언 표시**: 버튼 클릭 시마다 새로운 명언과 작성자 표시
- **중복 방지**: 연속으로 같은 명언이 나오지 않도록 함
- **부드러운 전환**: 텍스트 변경 시 페이드 인/아웃 효과

### 🎨 디자인 기능
- **라이트/다크 테마**: 🌙/☀️ 아이콘 버튼으로 테마 전환
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 최적화
- **카드형 레이아웃**: 중앙 정렬된 깔끔한 카드 디자인
- **애니메이션 효과**: 호버 효과, 버튼 애니메이션, 테마 전환 효과

### 💾 데이터 기능
- **로컬 저장소**: 테마 설정 자동 저장
- **시스템 테마 감지**: 사용자 OS 테마 설정 자동 감지
- **15개 명언 데이터**: 다양한 명언과 작성자 정보 포함

## 🛠️ 기술 스택

### HTML5
- 시맨틱 태그 사용 (`<header>`, `<main>`, `<section>`, `<footer>`)
- 접근성 고려 (`aria-label` 속성)
- 메타데이터 최적화 (`viewport`, `charset`)

### CSS3
- CSS 변수 시스템 (`:root`, `[data-theme="dark"]`)
- Flexbox 레이아웃
- 그라데이션 배경
- 백드롭 필터 효과
- 미디어 쿼리 (반응형)
- CSS 트랜지션 및 애니메이션

### JavaScript (ES6+)
- DOM 조작
- 이벤트 리스너
- 로컬 스토리지 API
- 시스템 테마 감지 (`prefers-color-scheme`)
- 배열 및 랜덤 함수 활용

## 🎯 학습 포인트

### 초보자를 위한 구조
1. **파일 분리**: HTML/CSS/JS 역할 분리 이해
2. **시맨틱 HTML**: 의미 있는 태그 사용법
3. **CSS 변수**: 테마 시스템 구현 방법
4. **DOM 조작**: JavaScript로 HTML 요소 제어
5. **이벤트 처리**: 사용자 인터랙션 구현
6. **반응형 디자인**: 다양한 화면 크기 대응

### 실용적인 기능
- **테마 전환**: 실제 웹사이트에서 많이 사용되는 기능
- **데이터 관리**: 배열을 활용한 데이터 처리
- **사용자 경험**: 부드러운 애니메이션과 피드백

## 🚀 실행 방법

### 로컬 실행
1. 프로젝트 폴더로 이동
2. Python이 설치된 경우:
   ```bash
   python -m http.server 8000
   ```
3. 웹 브라우저에서 `http://localhost:8000` 접속

### 파일 실행
- `index.html` 파일을 직접 브라우저에서 열어도 동작합니다.

## 🎨 디자인 특징

### 색상 시스템
- **라이트 테마**: 보라색 그라데이션 (#667eea → #764ba2)
- **다크 테마**: 어두운 남색 계열 (#1a1a2e → #16213e)
- **CSS 변수**: 일관된 색상 관리 시스템

### 레이아웃
- **중앙 정렬**: Flexbox를 활용한 수직/수평 중앙 정렬
- **카드 디자인**: 반투명 배경과 그림자 효과
- **반응형**: 모바일 우선 접근 방식

## 🔧 커스터마이징

### 명언 추가
`script.js` 파일의 `quotes` 배열에 새로운 명언 추가:
```javascript
{
    text: "새로운 명언 내용",
    author: "작성자 이름"
}
```

### 테마 색상 변경
`style.css` 파일의 CSS 변수 수정:
```css
:root {
    --bg-gradient-start: #새로운색상;
    --bg-gradient-end: #새로운색상;
    /* ... */
}
```
